Instalação do App no Wix

Pré-requisitos

  • Conta Wix com permissões de desenvolvedor
  • Acesso ao Wix Developer Center
  • Node.js (versão 14 ou superior)

1 Preparação do Ambiente Wix

  1. 1. Acesse o Wix Developer Center
  2. 2. Clique em "Create New Application"
  3. 3. Preencha as informações básicas:
    • Nome do App
    • Descrição
    • Categoria

2 Configuração do App

Permissões Necessárias:

  • Analytics.Read
  • Analytics.Write
  • Dashboard.Access

Credenciais:

App ID
App Secret Key

3 Integração com Pipeline Analytics

# Variáveis de Ambiente

PORT=8000
NODE_ENV=production
JWT_SECRET=seu_jwt_secret
WIX_APP_ID=seu_app_id
WIX_APP_SECRET=seu_app_secret

# Endpoints da API

/api/cards/month/:month  // Dados mensais
/api/cards               // Todos os cards

4 Instalação no Site Wix

No Wix App Market:

  • Publique seu app
  • Obtenha o link de instalação

No Site de Destino:

  • Acesse o Editor Wix
  • Add Apps > Seu App

Solução de Problemas

Erro de Conexão:

  • Verifique as credenciais
  • Confirme permissões OAuth

Dados não Aparecem:

  • Limpe o cache
  • Verifique a API

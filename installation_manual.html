Manual de Instalação

Pré-requisitos

  • Node.js (versão 14 ou superior)
  • NPM (Node Package Manager)
  • Navegador web moderno (Chrome, Firefox, Safari ou Edge)

Instalação do Sistema

1. Clone o Repositório

git clone [URL_DO_REPOSITÓRIO]
cd [NOME_DO_DIRETÓRIO]

2. Instale as Dependências

npm install

3. Configure o Ambiente

Crie um arquivo .env na raiz do projeto:

PORT=8000
NODE_ENV=development
JWT_SECRET=seu_jwt_secret

4. Inicie o Servidor

npm start

Configuração do Pipeline Analytics

Estrutura do Pipeline

O sistema de analytics está integrado com o seguinte pipeline de dados:

1. Coleta de Dados

  • Dados são coletados através da API REST
  • Endpoints principais:
    • /api/cards/month/:month - Dados mensais
    • /api/cards - Todos os cards

2. Processamento

  • Agregação de dados por período
  • Cálculo de métricas:
    • Total de oportunidades
    • Valor médio mensal
    • Taxa de crescimento
    • Taxa de conversão

3. Visualização

  • Gráficos interativos usando Chart.js
  • Dashboards personalizáveis
  • Filtros dinâmicos por:
    • Período
    • Produto
    • Agência
    • Responsável

Verificação da Instalação

  1. Acesse http://localhost:8000 no navegador
  2. Faça login com as credenciais de administrador:
    • Email: admin@sistema.com
    • Senha: admin123
  3. Verifique se o dashboard está carregando corretamente
  4. Teste a geração de relatórios analytics

Solução de Problemas

Erro de Conexão com o Servidor:

  • Verifique se a porta 8000 está disponível
  • Certifique-se que o Node.js está instalado corretamente
  • Verifique os logs do servidor no terminal

Problemas com Analytics:

  • Verifique a conexão com a API
  • Limpe o cache do navegador
  • Consulte os logs do console do navegador
